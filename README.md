# Sales-Performance-Dashboard

## Overview
This project is a Power BI dashboard created to analyze sales performance.

## Objective
To analyze sales data and create an interactive dashboard for better business insights.

## Features
- KPI cards (Total Sales, Profit)
- Bar chart, Pie chart, Line chart
- Filters (Slicers) for dynamic analysis

## Insights
- West region shows highest sales
- Technology category performs best
- Sales trend shows variation over time

## Tools Used
- Power BI
- Excel

## Dataset
Sample Superstore dataset used for analysis
